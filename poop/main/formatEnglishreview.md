## 语法

### 非谓语动词/谓语动词表示与判别

#### 谓语

+ were
+ had been

#### 非谓语

+ having
+ having been
+ to do, doing, done

#### 判别

1. 判断谓语和非谓语: 一句一谓语, 两谓一连词, 无连词用非谓语
2. 判断主动被动: 看逻辑主语 (上文), 若逻辑主语与主句主语一致可省略
    > 时态参照物: 与谓语动词的先后关系

### 反义疑问句

> 前肯后否, 前否后肯
>
> 构成: 助动词/be动词/情态动词+主语 (代词, 主格)

#### must 特殊情态动词

1. 表示肯定: must/mustn't -> need/needn't
2. 表示否定: mustn't -> must
3. 表示一定的疑问: must/must + 现在完成时 -> 与后面的词一致, 例如 must have /must have ... since last time -> haven't
4. 表示一定的疑问: must + 时间 -> didn't
5. be动词: am/is/are -> am/is't/aren't
