### shell 的 $

脚本调用参数标识符

- $0 Shell 本身的文件名

- $1～$n 添加到 Shell 的各参数值. $1 是第 1 参数, $2 是第 2 参数, $n 是第 n 个.

- $$ Shell本身的PID (ProcessID)

- $! Shell 最后运行的后台 Process 的 PID

- $? 最后运行的命令的结束代码 (返回值)

- $- 使用 Set 命令设定的 Flag 一览

- $* 所有参数列表. 若 "$\*" 用 _"_ 括起来的情况, 以"$1 $2 ... $n"的形式输出所有数.

- $@ 所有参数列表. 若 "$@" 用 _"_ 括起来的情况, 以"$1" "$2" ... "$n" 的形式输出所有参数.

- $# 添加到 Shell 的参数个数
